Project Story

OCR System: From Internship Build to Independent Extension

The OCR system began during internship work and later became a separate practice project to deepen image processing fundamentals.

Pipeline

The flow starts with image normalization and preprocessing, then character segmentation and extraction, and finally text output formatting. This approach improved consistency compared with direct raw-image extraction.

Processing stages

Input image → Preprocess/denoise → Segment regions → Extract characters → Return plain text

What Changed After Internship

The standalone version focused on code structure, repeatable test inputs, and better handling for low-quality text regions. The goal was to understand trade-offs, not just produce demo output.

Stack: Java, Image Processing, OCR pipeline concepts, ML basics
